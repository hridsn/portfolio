Train your Brain

Exercise game for Alzheimer's patients By urging the patients to make quick responses, they can improve their judgement skills. The data collected in this game can be used to study patterns and detect patients and give them the required treatment early.

About the project

Train your brain was a hackathon project that focused on the problems in the healthcare sector.

Through this project, we address the issue of Alzheimer's disease and pitch a game that can help with improving patient outlook on cognitive exercises.

Research

  • Sea Hero Quest - game for good. (2019, December 3). Deutsche Telekom. https://www.telekom.com/en/corporate-responsibility/corporate-responsibility/sea-hero-quest-game-for-good-587134

Proposed solution

A game that would encourage the users to play it more

  • Transport system to minimise graphics needed
  • Signs that made the user remember certain elements and use them later on
  • Something involving a track to make it easy for the users to follow and not get lost
  • Indicators to help the user if they are stuck

Concept Flow

Through the project

Splash screen

This screen gives the user a brief introduction to the game they will be playing. With the pun intended title, the user gets a slight amusement as well as an insight into what game this is.

Introductory screens

This screen explains the problem statement and the purpose of its development. The aim is to make the user understand the goals of their game and to inform them that their data in the game will be used for research purposes.

Research questions

These questions are asked to the user to analyse their rate of contracting Alzheimer's. The data obtained from reviewing the answers and the way they perform in each level can help identify early detection of the disease by matching patterns.

The Route Map

This interface shows the user the levels coming up ahead and also gives the provision to the user to select the level they want to play

Your Journal

This is where the user can find the map of the entire realm and choose where they would like to play.

This is also where they can check how they perform in the challenges presented to them in the game.

Game Map

Shows all the different areas where the game will be played from where the user can jump to any place provided it is available. Each area gets progressively difficult.



Performance Stats

Displays the performance index and other categories (out of 1000) of the player compared to other users playing the game.

Game Onboarding

A user-friendly method to introduce the elements of the game to the user in an interactive manner.

Zen Meditation Project

A personal design and development project to encourage getting acquainted with meditation.

About the project

When I landed my first job, I was quite stressed because of the environment change. On searching online, there were multiple breathing meditation exercises that I found as a remedy to this. But I never got around doing these exercises. This made me wonder how many others there would be like me who must deter from starting the exercises because they didn't know where to begin.

This made start working on the Zen Meditation Project, which would aid in bridging the gap between beginners and this vast world of meditation. By enabling the user to listen to music, giving step by step instructions on how to do a breathing exercise, and by not introducing them to multiple methods at once, I believe that anyone can start this journey of self care.


Note: These is currently no prototype for the project as it is still under development.
